Remember the milk (REMI)

Remi is a web application designed for Firefox OS devices that helps to not forget things.

With Remi you can manage lists of any kind. Choose a list name (e.g. Shopping list, to do, ...); add, edit, remove or check as 'done' list elements in a very simple and intuitive way.

Customize the main screen: choose from a set of page styles.

Remi is available on firefox os marketplace: click here.
